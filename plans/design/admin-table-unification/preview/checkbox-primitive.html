Checkbox primitive — visibility fix

Single-file shadcn override that cascades to every checkbox in the admin: row-selection checkboxes, type checkboxes (until D12 replaces them), and form-field checkboxes. Adds an explicit 2px slate-400 border at rest, fills with brand teal when checked, and renders a white check glyph at both 16px (row) and 20px (form) render sizes.

Token spec — Checkbox primitive override

TokenValueTailwind v4 classNotes
Border (unchecked) 2px solid · slate-400 border-2 border-slate-400 Replaces today's border border-border. Slate-400 reads on both white card and brand-muted row hover.
Border (checked + indeterminate) 2px solid · primary data-[state=checked]:border-primary data-[state=mixed]:border-primary Border color matches fill — the checkbox reads as one solid token.
Background (unchecked) background (white) bg-background Same surface as the row — the border carries the affordance.
Background (checked + indeterminate) primary (brand teal) data-[state=checked]:bg-primary data-[state=mixed]:bg-primary Filled selected state — F17/F18 fix.
Background (checked + hover) brand-dark data-[state=checked]:hover:bg-brand-dark Slight darken on hover signals interactivity.
Radius — sm 3px rounded-[3px] Subtle rounding — reads as a checkbox, not a square button.
Radius — md 4px rounded-[4px] Slightly more curve at form size to balance the larger surface.
Size — sm (row) 16 × 16px size-4 Used inside <RowActions> and table-header select-all.
Size — md (form) 20 × 20px size-5 Used inside form fields where it sits next to label text.
Check glyph — sm 10px lucide Check · stroke 3 · white size-2.5 stroke-[3] text-white Sized to clear the 2px border with even padding all around.
Check glyph — md 12px lucide Check · stroke 3 · white size-3 stroke-[3] text-white Scales with the box.
Indeterminate glyph lucide Minus · same dimensions as Check size-2.5 stroke-[3] text-white (sm) / size-3 (md) Used for select-all when 1+ but not all rows are checked.
Focus ring 3px brand / 25% opacity focus-visible:ring-3 focus-visible:ring-brand/25 Outside the box — visible on every focus state for keyboard nav.
Hover (unchecked) Border → brand hover:border-brand Color shift on hover — no background change at the unchecked stage.
Disabled opacity 50% · cursor not-allowed disabled:opacity-50 disabled:cursor-not-allowed Standard shadcn disabled treatment.
Transition background-color + border-color · 100ms ease-out transition-colors duration-100 ease-out Snappy state changes; no animation on the glyph itself.
