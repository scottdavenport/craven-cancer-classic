Components

One preview per shared primitive plus a summary tokens reference.

PR-1A · <RowActions>

Row actions — hover reveal

At-rest data only. On row hover: checkbox, edit pencil, delete trash, plus an optional surface-special slot (e.g. Teams "Mark paid"). Photos is exempt — D7.

PR-1A · <StatusTabs>

Status tabs with inline counts

Five surface variants. Active tab gets a brand-teal underline; inactive tabs use muted-foreground. Counts render as a subtle badge inline with the label.

PR-1A · <FilterBar>

Filter bar — search + dropdowns + chips

Three-row vertical structure: full-width search; uppercase-labeled secondary dropdowns; active-filter chips with a Clear-all link (hidden when 0 filters active).

PR-1A · <ModalSection>

Modal section — banded header + body

Uppercase section label sitting above a hairline rule. Body padding is generous enough to hold form-field grids without crowding.

PR-1A · D12 layout

Role cards — Contacts multi-type form

Five always-visible role cards. Selected cards expand inline; unselected cards stay collapsed at ~55% opacity. No layout shift on toggle. Sample contact: Lynne Davenport.

PR-1B · <Checkbox> override

Checkbox primitive — F17 / F18 fix

Visible 2px slate-400 border at rest. Filled brand-teal background + white check when selected. Both row-selection (16px) and form (20px) sizes verified.
