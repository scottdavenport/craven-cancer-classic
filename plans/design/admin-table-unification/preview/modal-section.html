<ModalSection>

Banded section header (uppercase label + horizontal hairline rule) followed by a body region that holds form fields. Stacks inside any shadcn <Dialog>. Used everywhere — even short forms (Teams: Roster + Payment) get banding for IA consistency. Photos has no edit modal so it's exempt from this primitive.

Three stacked sections inside a Dialog

Identity → Roles → Address. Each one renders the same primitive — only the label and body content vary.
